
# [web\website\blog_1.html](../../web/website/blog_1.html)

## 设计特点：

**1. 标题和描述：**
- 渐变色动画标题"我的知识花园"
- 富有诗意的副标题描述
- 大型英雄区域展示

**2. 背景图：**
- 使用高质量的书籍/学习主题背景图
- 添加了渐变遮罩提升可读性
- 支持视差滚动效果

**3. 卡片导航：**
- 6个主题卡片（前端、后端、算法、设计、工具、思考）
- 每个卡片都有独特的图标和颜色
- 悬停效果和点击导航功能
- 响应式网格布局

## 特色功能：

- **视觉效果**：渐变背景、卡片阴影、悬停动画
- **交互体验**：点击导航、平滑滚动、进入动画
- **响应式设计**：适配手机和桌面设备
- **现代化UI**：玻璃态效果、微交互动画

## 技术实现：

- 纯HTML/CSS/JavaScript，无需额外依赖
- 使用CSS Grid和Flexbox布局
- CSS3动画和过渡效果
- JavaScript处理交互和导航
