### 3D 卡片动画生成器 - 3d-101.html

**功能概述**

一个高性能的 3D 卡片动画生成工具，支持上传任意图片，实时生成具有透视感和质感的卡片翻转动画。用户可以通过可视化参数调整实时预览动画效果，支持多种预设风格和自定义组合。

---

**视觉风格**

高对比度界面：文本采用高亮紫色（#7c6aff）和粉红色（#ff6a9c）作为强调色，确保视觉聚焦。

流畅动画设计：整体交互采用缓动曲线和 3D 透视效果，呈现高质感的物理运动感受。

现代化卡片呈现：上传的图片以圆角卡片形式展示，融合投影、高光和暗角效果，增强空间感和深度感。

**界面分区**
顶部区域：品牌标识（✦ 3D ANIMATOR）+ 主标题 + 引导文字

上传区域：交互友好的拖拽上传区，支持点击触发文件选择

控制面板：栅格式参数调整区域，包含 5 个快速预设 + 10+ 个细化参数

预览舞台：沉浸式动画演示区，具有透视网格背景和阴影投影

时间轴：实时显示动画播放进度和当前阶段（入场 → 停留 → 退出）

## 核心功能

**1. 图片上传与加载**
- 支持格式：JPG、PNG、WebP、GIF
- 上传方式：拖拽或点击触发文件选择器
- 建议比例：16:9（自适应缩放）
- 加载后自动进入编辑模式

**2. 5 大预设动画风格**

| 预设名称 | 形容 | 用途 |
|---------|------|------|
| **轻柔入场** | 温和、优雅的 3D 翻转，低透视强度 | 品牌发布、柔和展示 |
| **戏剧翻转** | 大幅旋转、高透视距离短，戏剧张力强 | 产品宣传、吸引注意 |
| **侧滑展开** | Y 轴小角度翻转，伴随水平滑动感 | 卡片列表、渐进式加载 |
| **向上升起** | X 轴高角度翻转 + 垂直位移，浮动感强 | 升级通知、成就展示 |
| **弹性弹出** | 初始透明 + 弹性缓动，活泼有力 | 游戏化界面、互动反馈 |

**3. 可视化参数调整**

所有参数支持实时预览，共 12 个可调选项：

- **Y 轴旋转**（-60° 到 60°）：控制左右翻转角度
- **X 轴旋转**（-30° 到 30°）：控制上下倾斜角度
- **初始缩放**（60% 到 100%）：入场时卡片大小
- **初始透明度**（0% 到 100%）：入场时卡片透明度
- **动画时长**（0.2s 到 3.0s）：入场动画持续时间
- **透视距离**（300px 到 2000px）：3D 透视强度（越小越强）
- **缓动曲线**：预置 6 种缓动函数供选择
- **暗角强度**（0% 到 100%）：卡片边缘阴影强度
- **停留时长**（0s 到 8.0s）：卡片静止展示时间
- **退出时长**（0.1s 到 2.0s）：退出动画持续时间
- **退出方向**：支持向右翻出、向左翻出、向上淡出、向下淡出、缩小消失、无退出
- **循环播放**：单次播放或自动循环

**4. 动画执行流程**

```
重置状态（0ms）
    ↓
等待（50ms）
    ↓
入场动画（指定时长）：从初始状态 → 正面平视（0°/0°/100%）
    ↓
停留阶段（指定时长）：卡片静止展示，暗角逐渐加强
    ↓
退出动画（指定时长 OR 无）：根据选择方向消失
    ↓
[循环] 或 [结束]
```

**5. 实时反馈系统**

- 关键帧预览：显示 CSS 动画序列、总时长、各阶段描述
- 时间轴进度条：实时进度条 + 阶段标识（▶入场 → ⏸停留 → ◀退出 → ✓完成）
- 参数值显示：滑块旁实时显示当前值（单位：°、%、s、px）
- 质感增强：高光、投影、网格背景共同营造沉浸感

## 技术亮点

**性能优化**
- 使用 CSS 3D Transforms 实现硬件加速动画
- will-change 优化、显式分层处理
- 高效的事件委托和防抖逻辑

**动画精度**
- 支持三阶贝塞尔曲线缓动，精细控制动画节奏
- animation-duration、animation-delay 精确控制
- requestAnimationFrame 驱动的时间轴同步

**用户体验**
- 实时预览：任何参数变更立即反映在舞台上
- 快速预设：一键应用完整动画配置，快速原型设计
- 响应式设计：支持各屏幕尺寸，移动端友好
