# 模板更新通用指令

## 当制作新模板时，执行以下操作：

### 1️⃣ 更新 README.md
- **位置**：在对应类别的表格行中添加新条目
- **格式**：`| 类别 | [文件名](./web/分类/文件名.html) | 简短说明 |`
- **示例**：`| 母婴卡片 | [xhs01.html](./web/xhs_card/xhs01.html) | 温馨母婴知识卡片 |`

### 2️⃣ 更新 index.html（如适用）
- 在对应分类区域添加新模板卡片
- 包含链接、缩略图和描述

### 3️⃣ 提交代码
```bash
git add .
git commit -m "新增：母婴卡片模板 xhs01.html"
git push origin main
```

### 4️⃣ 自动部署
- GitHub Actions 会自动部署到 GitHub Pages
- 在线访问：`https://tuonioooo.github.io/fast-web-template/web/xhs_card/xhs01.html`
